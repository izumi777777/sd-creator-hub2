**機能説明 × 技術解説のシリーズ**（日付記事はそのまま残し、分解用の下書き）は [drafts/series/README.md](drafts/series/README.md) を参照。

## 運用メモ

- 下書きは **Markdown**。`drafts/images/` に挿絵 PNG がある記事は、**note では「画像を追加」からアップロード**する（Markdown の `![](...)` は note 側ではそのままでは効かないことが多い）。
- **アイキャッチ**用の `drafts/images/2026-04-04_eyecatch.png` や `2026-04-05_eyecatch.png` などは、note の **アイキャッチ設定** に指定する（本文画像とは別枠）。
- **個人情報・APIキー・バケット詳細**は書かない。公開前に必ず再確認する。
