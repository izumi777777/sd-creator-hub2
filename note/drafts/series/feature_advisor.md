# 【機能】全ページ共通の Gemini 相談ドック【下書き】

## この画面でできること

- どのページからでも（ドック UI がある場合）**会話形式で Gemini に相談**できる。
- **相談モード**が URL に応じて初期値が変わり、プロンプトの前提が切り替わる（ストーリー画面ならストーリー向け、など）。
- **ストーリー詳細**では、保存済み本文・章・プロンプト抜粋を **「ストーリー資料」** としてシステム側に添付でき、**Pixiv のタイトル・キャプション・タグ**などをストーリーに沿って依頼しやすい。
- **一覧で生成した直後**は「相談ドックに渡す」でセッションに載せ、同じく資料として使える。
- **会話リセット**で履歴とドラフト資料を消せる。

---

## 使い方のコツ

- 資料を含めるかは **チェックでオン／オフ**。長い作品は自動でトリムされる前提なので、必要なら依頼文で「第◯章中心に」と補足。
- 単発の販売文だけなら **テキスト生成**ページ、**会話で詰めたい**ときは相談ドック、と使い分けると整理しやすいです。

---

## おわりに

「同じ話を二回説明しない」ための UI です。公開記事では **API キーやプロンプト全文**は載せないでください。
